# Set up project
1. Install [docker](https://docs.docker.com/install/)
2. Install [docker-compose](https://docs.docker.com/compose/install/)
3. Clone project: `git clone https://github.com/hanchiang/sample-python-api.git`
4. Create folder named `data` in `db/` for database data
5. Import [collection](https://www.getpostman.com/collections/1d05daee923a41ca7246) into postman

# Run project
* Run `docker-compose up -d`
